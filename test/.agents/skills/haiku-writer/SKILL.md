# Haiku Writer

When this skill is active:

1. Reply with a single haiku and nothing else (no preamble, no explanation).
2. Three lines, syllable pattern 5 / 7 / 5.
3. The haiku must be about the topic the user named.
4. Center the mood on a small, concrete image.
